
Tech Stack – Fitness Tracking Application

Frontend
    Angular
    Bootstrap
    ng2-charts

Backend
    ASP.NET Core Web API
    Repository Pattern
    
Database
    PostgreSQL
    Accessed via EF Core (Code-First ORM)

Authentication & Authorization
    JWT (JSON Web Tokens) for authentication
    RBAC (Role-Based Access Control) implemented using ASP.NET Core’s

Logging
    Serilog for structured logging (console and file output)

Notification
    SignalR

App Hosting Runs locally during development
Tested on Azure Virtual Machine, Tested in Docker Containers

Cloud Services 
    AWS S3 Bucket used for storing images
